# react-vite-boilerplate

React+Vite Boilerplate

# Requirement

- Docker
- VSCode
  - Dev Container Extension

# Usage

## Server Side

Open `server` directory in Dev Container and

```sh
deno task start
```

## Client Side

Open `client` directory in Dev Container and

```sh
yarn
yarn dev
```

# Testing

## Server Side

```sh
deno task test
```

## Client Side

```sh
yarn test
```
